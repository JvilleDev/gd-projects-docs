## Descripción del proyecto
Usando el Webkit de Google Chrome, transcribe en tiempo real la señal entrante por parte del micrófono seleccionado en el sistema, y posteriormente envía el resultado a través de Socket para su visualización.

Este proyecto también se compone de tres páginas:
- Página para transcripción
- Página para ver la transcripción con búsqueda incluida
- Página para agregar la transcripción al programa
