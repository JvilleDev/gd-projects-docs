## Descripción general del proyecto
El proyecto se compone de 3 páginas sencillas, las cuales se irán viendo a mayor profundidad a medida que se avanza.
### Página principal
![](/himnario/home-page-song-selected.webp)

Esta página esta compuesta por 3 elementos (Excluyendo la barra de navegación):
- **Elemento A:** Seleccionar canto.
- **Elemento B:** Cambiar de línea/verso.
- **Elemento C:** Buscar por título o número.
---

### Página para modificar lista
![](/himnario/edit-page.webp)

Esta página también está compuesta por 3 elementos:
- **Elemento A:** Tabla que muestra los himnos actualmente con la información pertinente.
- **Elemento B:** Cada fila corresponde a un himno, el cual a su vez posee ciertas acciones que puede realizar, las cuales son editar o eliminar dicho canto.
- **Elemento C:** Botón para agregar un himno.

---

### Página para mostrar las líneas entrantes
Esta página es más sencilla que las anteriores, pues sólo se encarga de recibir información. Dependiendo del enlace con el que se acceda cambiará su estilo. Esto se verá después en la sección dedicada.
